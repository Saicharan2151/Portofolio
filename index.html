Title: Sales Tax Calculator

Description:

The "Sales Tax Calculator" is a web application designed to assist users in calculating sales tax and the total cost of a purchase. This simple and user-friendly tool enables users to input the subtotal of their purchase and the applicable tax rate, then computes the corresponding sales tax and total cost, providing an instant breakdown of the financial aspects of a transaction.

Technologies Used:

HTML:

HTML shapes the structure of the webpage, incorporating labels and input fields for user input, along with headings and buttons to facilitate user interaction.

CSS:

To enhance the visual appeal and elevate the user experience, the page is styled using an external CSS file named sales.css.

JavaScript:

TJavaScript is employed to execute the logic behind computing sales tax and presenting the results. The code is distinctly separated from the HTML for enhanced maintainability and organization, achieved by incorporating an external JavaScript file named "sales_tax.js."

Title: Test Score Array

Description:

The web application known as "Test Score Array" is crafted with the purpose of aiding users in the organization and analysis of test scores. By allowing users to input both the student's name and corresponding test score, this user-friendly tool facilitates the addition of data to an array. Users can effortlessly display the results and review the entered scores, providing a straightforward system for recording and organizing test scores.

Technologies Used:

HTML:

The structure of the HTML encompasses input fields for the student's name and test score, accompanied by buttons for both adding data to the array and displaying the results. This page structure is elevated through the thoughtful inclusion of labels and buttons, enhancing the overall user experience.

CSS:

The project enhances its visual appeal and elevates user experience by incorporating an external CSS file, text_score.css, dedicated to styling. This file applies aesthetic styles to diverse elements within the web page, contributing to an overall improved user interface.

JavaScript:

JavaScript takes the reins in managing the test score array, handling data addition, and presenting results. The implementation unfolds through an external JavaScript file, namely "test_scores.js," strategically included to uphold a codebase that is neat and well-organized, distinct from the HTML structure. Tasked with user interactions and data manipulation, JavaScript showcases its prowess in steering these crucial aspects of functionality.

Title: Complete our survey

Description:

The web page titled "Take Our Survey" is meticulously crafted to collect user input through a well-organized form. Featuring a user-friendly interface, it guides users through the survey process effortlessly. The form encompasses diverse fields, including contact information, geographic details, and a segment dedicated to specifying how users came across the survey. The primary goal of this page is to simplify the data collection process, guaranteeing accurate completion of essential fields. In essence, it serves as a highly efficient tool for aggregating valuable user responses.

Technologies Used:

HTML:

The form's structure is outlined in HTML, delineating its different input fields. Attributes like "autofocus" and "required" have been incorporated to enhance the user experience. "Autofocus" directs attention to the email input, while "required" ensures that crucial fields are filled out for a smoother interaction.

CSS:

Employing embedded CSS, the project intricately tailors the form's aesthetics, crafting an attractive and neatly arranged layout. It precisely defines font styles, field dimensions, and form validation styles, enhancing the visual appeal and user-friendliness of the interface.

Form Validation:

To enforce data validity, HTML attributes such as "required" and "pattern," along with specific input types, are utilized for form validation. Furthermore, the inclusion of checkboxes serves the purpose of gathering information on how users came across the survey.
